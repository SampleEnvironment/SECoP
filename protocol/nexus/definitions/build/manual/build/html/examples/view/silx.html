2.2.2. Plot a NeXus HDF5 file with silx view¶

A NeXus HDF5 file with plottable data (see Find plottable data in a NeXus HDF5 file) can be plotted by the silx view [1] tool provided as part of silx [2].

[1]

silx view : http://www.silx.org/doc/silx/latest/applications/view.html

[2]

silx : http://www.silx.org/doc/silx/latest/
