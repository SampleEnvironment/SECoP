NXxbase
=======

**Status**:

  application definition, extends :ref:`NXobject`

**Description**:

**Symbols**:

  The symbol(s) listed here will be used below to coordinate datasets with the same shape.

  **nP**: Number of points

  **nXPixels**: Number of X pixels

  **nYPixels**: Number of Y pixels
